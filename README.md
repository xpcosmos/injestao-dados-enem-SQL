# Introdução

Esse projeto tem o objetivo de estruturar dados do enem em bancos de dados e analisar os dados utilizando métodos estatísticos. 

Conhecimentos adquiridos/aprimorados durante o projeto:

* Spark
* SQL
* Python
* Bash/Linux/Unix
* Modelagem de dados
